La aplicación utiliza la API pública Logotypes.dev, un servicio open-source que proporciona logotipos de marcas y empresas de todo el mundo en formatos vectoriales (SVG) y, en algunos casos, en formato PNG.

Funcionalidades de la aplicación
La aplicación fue desarrollada en Python utilizando Tkinter como interfaz gráfica y ofrece las siguientes funciones:
Permite al usuario ingresar el nombre de una marca.
Genera automáticamente la URL del logotipo correspondiente en Logotypes.dev.
Abre el logotipo en el navegador web, mostrando la imagen disponible (principalmente SVG).
